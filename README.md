# Doctor License Management System

## Backend
- .NET 8 Web API
- Clean Architecture
- CQRS pattern
- SQL Server
- Dapper + EF Core

## Features
- Create, Update, Delete (Soft delete)
- Search & filter doctors
- License expiry logic (auto Expired)
- Stored procedure for listing
- Standard API response structure

## How to run
1. Open backend solution in Visual Studio
2. Update connection string in appsettings.json
3. Run SQL scripts in `/database`
4. Run API
5. Open Swagger: https://localhost:xxxx/swagger
